# Mandvi Singh – Portfolio

A professional single-page portfolio built with HTML, CSS, and JavaScript.

## Setup

1. **Resume (PDF)**  
   Place your resume PDF in this folder and name it **`resume.pdf`**.  
   The "Download Resume" buttons will then offer this file for download.

2. **Personalize**  
   Update in `index.html`:
   - Email: `mandvi.singh@example.com`
   - LinkedIn, GitHub, Twitter URLs
   - Project and blog links
   - Certificate names and links (if any)

3. **Run locally**  
   Open `index.html` in a browser, or use a local server:
   ```bash
   # Python
   python -m http.server 8000
   # or Node (npx serve)
   npx serve
   ```
   Then visit `http://localhost:8000`.

## Sections

- **Hero** – Name and short intro (third-year B.Tech CSE, DSA focus)
- **About Me** – Bio and highlights
- **Skills & Technologies** – Languages, DSA, web, tools
- **Projects** – Project cards with tags and links
- **Certificates & Courses** – List of certifications/courses
- **Experience** – Timeline (education and roles)
- **Blogs** – Blog cards with placeholders
- **Contact** – Email, LinkedIn, GitHub, Twitter + resume download

## Files

- `index.html` – Structure and content
- `styles.css` – Layout and styling
- `script.js` – Nav toggle, scroll behavior, footer year, resume download
